# Tillgänglighetsredogörelse — mall (per kommun)

[Kommunen] tillhandahåller UBM Klar för intern ärendehantering. Vi vill att så många som
möjligt ska kunna använda tjänsten. Denna redogörelse beskriver hur tjänsten uppfyller
lagen om tillgänglighet till digital offentlig service, kända brister och hur du kan
rapportera problem.

## Efterlevnadsstatus

Tjänsten är **helt/delvis** förenlig med WCAG 2.1 nivå AA och EN 301 549.

## Tekniska tillgänglighetsegenskaper

- Fullständig tangentbordsnavigering med synlig fokusmarkering och hopp-länk
- Semantisk struktur: landmärken, rubriknivåer, tabellrubriker, formulärkoppling
- Statusinformation med både färg och text; AA-kontraster i designsystemet
- Tydliga laddnings-, tomt-, fel- och behörighetslägen på alla sidor
- Svenskt klarspråk; felmeddelanden förklarar vad som hänt och vad man gör härnäst
- Maskerade fält annonseras för skärmläsare med instruktion om skälkrav
