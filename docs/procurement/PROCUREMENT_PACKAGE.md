# Procurement Package

This folder must contain the documents municipalities need for procurement, security review, legal review and cloud/outsourcing assessment.

## Required documents

- product description
- deployment model responsibility matrix
- Model B description
- Model C description
- support access model
- subprocessor list
- data residency statement
- encryption and key management statement
- incident process
- backup/restore responsibility
- exit process
- SLA appendix
- security appendix
- PUB/DPA appendix
- DPIA support document
- accessibility statement
- archive/retention approach
- municipality-owned data plane manual

## No-PII rule

Procurement and support material must never include citizen personal data, documents, UBM payloads, control case content or case notes.
